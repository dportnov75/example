Пример проекта.

В проекте представлен функционал двух групп акторов:
1. Директора школ
2. Директора департаментов образования

Функционал не имет отношение к реальности

Т.к. в проекте используется MapStruct, то для успешного запуска, сначала необходимо выполнить 
«mvn clean package», в обратном случае не будут созданны реализации интерфейсов, 
занимающихся преобразованием сущностей в модели доменного слоя и обратно.

Так же, в среде должна быть установлена поддержка Lombok не ниже v 1.8.4

Покрытие тестами не 100%, даже и не 5%

Стэк: Spring Boot, GraphQL, Lombok, MapStruct, Spring Data, Spring Cloud Stream, AspectJ, Hateoas, Zipkin
